# 🌦️ Weather Dashboard (React.js)

A simple, responsive Weather Dashboard built with React.js.  
Fetches real-time weather data from OpenWeather API.

## 🚀 Features
- Search for any city
- Current weather (temperature, humidity, wind, description)
- 5-day forecast
- Temperature toggle (°C ↔ °F)
- Dark mode
- Responsive design
- Last searched city persists (localStorage)

## 🛠️ Tech Stack
- React.js
- JavaScript (ES6+)
- CSS / Tailwind (optional)
- OpenWeather API
- Deployment: Vercel / Netlify
